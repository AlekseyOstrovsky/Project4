Поиск репозиториев github

Осуществляет поиск репозиториев по введенной поиском подстроке. Выводит первые 10 результатов.

Например, tetris+language:assembly&sort=stars&order=desc осуществляет поиск репозиториев на ассемблере по ключевому слову "Tetris" с сортировкой по убыванию "звезд", т.е. наверху окажутся наиболее популярные.
